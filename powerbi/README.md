Global Sales Performance Dashboard (Power BI)
Overview

This Power BI dashboard analyzes global sales data to highlight key trends in revenue, profitability, regional performance, and product contribution. It is designed to support business decision-making through clear KPIs and interactive visuals.

Tools

Power BI Desktop

DAX (Data Analysis Expressions)

Dashboard Features

KPI cards: Total Sales, Total Profit, Profit Margin, Total Orders

Monthly sales trend analysis

Sales performance by region

Profitability by product category

Top 10 products by sales

Interactive slicers for dynamic filtering

Data Model

A dedicated Date Table was created and linked to the sales data to enable accurate time-based analysis and stable reporting.

Key Insights

Revenue growth varies significantly by region

Profitability differs across product categories

A small number of products contribute a large share of total sales
